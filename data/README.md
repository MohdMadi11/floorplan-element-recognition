# Data Folder

Place assignment PDFs in this folder before running the pipeline.

The PDF files themselves are intentionally ignored by Git because floor plans may be private or company-provided. The expected local filenames used by the VS Code tasks are:

- `floorplan1.pdf`
- `floorplan2.pdf`
- `floorplan3.pdf`
- `floorplan4.pdf`
- `company_test.pdf`
